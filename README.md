# ski-in-singapore
RedMart challenge for Ski In Singapore

### Usage:
```
Ski Map - Longest Run.
Usage:
  skimap [--input=<file>]
  skimap -h | --help
  skimap --version

Options:
  -h --help       Show this screen.
  --version       Show version.
  --input=<file>  File to Parse [default: map.txt]
```

Challenge response in Go for RedMart's "Skiing in Singapore - a coding diversion"
